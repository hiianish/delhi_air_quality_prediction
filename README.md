# Delhi Air Quality Prediction

A machine learning project to analyze and predict air quality in Delhi using historical pollution data.

## Overview
- Performs exploratory data analysis on air quality data
- Cleans and preprocesses the dataset
- Trains regression-based machine learning models
- Analyzes errors and feature importance

## Files
- `AQI DATASET.csv` – raw dataset  
- `clean_dataset1.csv`, `final_dataset.csv` – processed data  
- `exploratory_data_analysis.py` – EDA  
- `data_preprocessing.py` – data cleaning  
- `model_training.py` – model training  
- `error_analysis.py` – evaluation  
- `Feature_Importance_(Interpretation).py` – interpretability  

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## Goal
Predict air quality values and understand the impact of different pollutants.
